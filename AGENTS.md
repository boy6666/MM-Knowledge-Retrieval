# AGENTS.md

Agent configuration for this repository. Skills and conventions used by coding agents when working on this project.

## Agent skills

### Issue tracker

Issues and external PRs are tracked in GitHub Issues. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context — `CONTEXT-MAP.md` at root maps to `backend/CONTEXT.md`, `ai-core/CONTEXT.md`, and `frontend/CONTEXT.md`. See `docs/agents/domain.md`.
